# AntiScam-Callguard🛡️

一款完全本地运行的安卓应用，实时分析通话内容并识别诈骗风险。

## 功能特性
- 📞 **自动监听来电**：通话时自动启动录音  
- 🔒 **隐私保护**：所有数据处理均在本地完成  
- 🚨 **实时提醒**：检测到风险关键词时震动并弹窗  
- 📂 **通话历史**：查看录音、文字内容和检测结果  

## 技术栈
- **前端**: Android (Kotlin)  
- **语音识别**: [Vosk离线引擎](https://alphacephei.com/vosk/)  
- **AI模型**: TensorFlow Lite (预训练诈骗关键词分类模型)  
- **本地存储**: SQLite + 文件系统
